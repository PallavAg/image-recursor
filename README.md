## About

**Gartic phone for images using AI.**

This website uses OpenAI's DALL路E 3 and GPT-4 Vision API. The starting prompt generates an image using DALL路E 3, then GPT-4V is used to describe the image. This description is used as the prompt for the next DALL路E 3 generation.

[![image](https://github.com/PallavAg/image-recursor/assets/40081309/7faa3b77-dbe8-4874-9bac-31ab2070c7df)](https://image-recursor.web.app/)

## Running the project
### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
